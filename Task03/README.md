# Task03 — «Простое ли число?»

Веб-приложение (SPA) — игра, в которой игроку предлагается случайное целое число, и он должен определить, является ли оно простым.

## Запуск приложения

### 1. Установить зависимости

```bash
cd Task03
composer install
```

### 2. Запустить локальный веб-сервер

```bash
php -S localhost:3000 -t public
```

### 3. Открыть в браузере

```
http://localhost:3000/
```

База данных `db/prime.db` создаётся автоматически при первом запросе.

## Требования

- PHP >= 8.0 с расширениями `pdo`, `pdo_sqlite`
- Composer
