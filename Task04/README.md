# Игра «Простое ли число?»

Веб-приложение на фреймворке **Laravel**. Игроку предлагается случайное целое число, он должен определить, является ли оно простым.

## Описание игры

- Игроку выдаётся случайное целое число (от 2 до 200).
- Игрок вводит своё имя и выбирает ответ: «Простое» или «Составное».
- После ввода программа сообщает, правильный ли ответ.
- Если число составное — дополнительно выводятся его нетривиальные делители.
- Все игры сохраняются в базе данных (SQLite).
- Доступна страница просмотра истории всех игр.

## Требования

- PHP >= 8.2
- Composer
- SQLite (встроен в PHP)

## Установка и запуск

### Автоматическая установка (рекомендуется)

```bash
make install
```

Эта команда автоматически выполнит все необходимые шаги:
1. Установит зависимости (`composer install`)
2. Создаст файл `.env` из `.env.example`
3. Сгенерирует ключ приложения (`php artisan key:generate`)
4. Создаст файл базы данных SQLite (`database/database.sqlite`)
5. Выполнит миграции (`php artisan migrate`)

### Ручная установка

```bash
# 1. Установить зависимости
composer install

# 2. Создать файл конфигурации
cp .env.example .env

# 3. Сгенерировать ключ приложения
php artisan key:generate

# 4. Создать файл базы данных SQLite
touch database/database.sqlite

# 5. Выполнить миграции
php artisan migrate
```

### Запуск веб-сервера

```bash
make serve
# или
php artisan serve
```

После запуска откройте в браузере: **http://localhost:8000**

## Структура приложения

| URL | Описание |
|-----|----------|
| `/` | Главная страница — форма для игры |
| `/game/play` | Обработка ответа и вывод результата (POST) |
| `/history` | История всех игр |

## База данных

БД SQLite расположена в `database/database.sqlite`.

### Таблицы

- **players** — игроки (id, name, created_at, updated_at)
- **games** — игры (id, player_id, number, is_prime, user_answer, is_correct, divisors, played_at)
